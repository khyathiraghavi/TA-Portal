Applicant Tab

Select Applicants

  • select applicants for a particular course
  • students are color coded according to their properties as shown on the page
  • students can be selected for quarter,half,full taships
  • Reset option is provided if all current selections are to be rest

Applicant Status

  • Enter name of Applicant and check his/her status of application

Selected Applicants

  • List of Applicants according to selected filters
  • Filters are By Course,By Name,By Roll No.,By Email Id,By Program of Study

Ta's Tab

Unselect Ta

  • Select course from which Ta's are to be removed
  • Ta's selected will show up and Ta's which are to be removed can be removed

Allocated Ta's

  • The required number of Ta's(Quarter,Half,Full) to be allocated for each course

Courses Tab

Add Faculty

  • Add faculty to existing course

Allocate Program

  • Allocate course for a particular program of study(CLD,CSE,CSD etc.)

Course Info

  • Information related to all the courses currently offered

Add course

  • Add course to the list of offered courses

Update course

  • Choose a course from the given list
  • On selection the current details of the course will be shown
  • Thsese details can then be edited and submitted

Delete

  • Course,Faculty,Program can be deleted

Settings Tab

Set Date

  • This will show the start and end dates for applications of Taship by students
  • Students cannot apply for taship before or after the deadlines
  • If dates have not been set they will not be displayed
  • Below the display of dates option of changing the dates is given

Update db

  • This functionality allows user to add multiplpe course and their respetive details
  • The format of the file(.csv) is described on the page itself

Log Table

  • This functionality allows user to see all activity on the portal
  • There are filters available looking up for specific activity(loggin in,selecting ta's.applying for taship etc.)

Upload Student Info

  • This functionality allows user to upload ISAS dump for all the students so as to verify and validate their credentials
  • In case data not present in the dump, applicants will have to manually fill up their profile information.
  • Format for dump will be Name, Roll Number, EmailID, CGPA separated by ','

Upload Feedback

  • This functionality allows user to upload feedback from a dump file for the students who have been TA's in past years
  • Comments and ratings are given in by faculty under whom they were acting as TAs
  • Format of the dump will be Name; Roll Number; Course ID; Rating(given by faculty); Course Name; Semester in which they were TA; and faculties comments about their work separated by ';' (semi colon) .

View Previous Applicant Log

  • This functionality allows user to see all the previous years Applicants.

Notifications Tab

  • View of specific activity on the portal
  • The activity is divided based on time of activity

  • A list of courses will be provided.
